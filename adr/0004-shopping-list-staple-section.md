# 买菜清单分出「常备调料」栏：按常备表自动归类

## Context

选好菜后，盐 / 糖 / 油 / 酱油这类家里常备的调料不必每次买，混在主清单里反而干扰「这次真正要买什么」。

## Decision

买菜清单自动分两栏：**主清单**（这次要买）+**常备调料**。归类用一张**可扩充的常备表**按规范名精确匹配（不做子串，避免误判），与 [adr/0002](0002-shopping-list-amount-aggregation.md) 的 `GOOD_CANON` 同风格。葱姜蒜这类新鲜辛香料**按用户选择归入常备**。分类是派生的、不额外存储；勾选 / 持久化机制不变。

## Consequences

- 归类不准时改常备表即可；v1 不引入「每食材用户标记」这类更重的机制。
- 复制导出的文本清单也分「主清单 / 常备调料」两段。
